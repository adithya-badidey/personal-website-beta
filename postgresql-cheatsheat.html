PostgreSQL Cheatsheat

  • To list Databases
    \list
  • To change database
    \c <db_name>
  • To list tables in database
    \dt
  • To show table schema
    \d+ <table_name>
  • To quit
    \q
  • To execute .sql file
    psql -U username -d myDataBase -a -f myInsertFile
  • Resize column without losing data
    • http://sniptools.com/databases/resize-a-column-in-a-postgresql-table-without-changing-data
